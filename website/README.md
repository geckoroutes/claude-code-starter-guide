# Claude Code Starter Guide — Website

Companion website for the starter guide. Deployed to Vercel at [claude-code-starter-guide.vercel.app](https://claude-code-starter-guide.vercel.app).

## Stack

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4 + Framer Motion
- Lucide React icons

## Pages

- `/` — Landing page (hero, how-it-works, animated demo, build cards, CTA)
- `/setup` — Setup wizard (3-step: Accounts, System, Install)
- `/start` — Getting started (starter prompts, level-up suggestions)

## Development

```bash
cd website
npm install
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000).
